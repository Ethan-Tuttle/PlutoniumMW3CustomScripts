To run the scripts
=====================
Make sure plutonim is installed ( duh )

Open Run.exe >

Type %APPDATA% > 

Go to AppData Folder ( parent folder ) >

Go to Local >

Go to Plutonium >

Add folder called scripts ( if it doesnt exist ) >

Paste these script files into that folder
